微软：Windows 10预览版手机可回滚至WP 8.1

16 威锋网   2015-02-11 07:27:41

摘要：微软 Windows Insider 项目负责人 Gabriel Aul 此前曾多次暗示，微软可能会在 Windows 10 手机预览版中添加一 项非常实用的新功能——系统降级回滚功能。尽管这个功能设想对用户而言十分有用，但最终是否会落实到预览版中依然是个未知数，不过微软这位“发言人”今天 正式确认了此事。


根据外媒 WMP 最新报道，Gabriel Aul 今天在回复网友提问时正式宣布，微软将在推送 Windows 10 手机预览版时正式上线这项全新的功能。Gabriel Aul 在推文中原话称：“是的，我们将会推出一款恢复工具来支持所有更新预览版的设备进行系统降级回滚”。这也就意味着，未来更新到 Windows 10 手机预览版的用户将可以自由刷机回 Windows Phone 8.1 系统。

考虑到近日有报告指出，当前正在微软内部进行测试的 Windows 10 手机技术预览版存在一些重大 Bug，包括设备掉电快、易发热、动画效果不流畅等诸多问题，而且预览版在推出公测期间若还出现其他意想不到的问题，那么系统回滚功能对用户来说将会非常重要。

当然，我们现在最希望的肯定是微软在推出这项回滚工具之前能够在内测阶段彻底解决上述提到的系统问题。毕竟谁都不希望在更新到新一代 Windows 10 手机预览版系统之后，还要去体验一下这个新的回滚功能。
