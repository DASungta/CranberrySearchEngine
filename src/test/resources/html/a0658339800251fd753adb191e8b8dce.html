微软：Denim更新并非升级Windows 10必要条件

10 威智网   2015-02-11 13:49:45

摘要：2月11日消息，虽然微软已经表示 Windows 10 手机预览版将会在本月到来，但我们现在依然还不清楚具体将在何时正式推送。不过，微软 Windows Insider 项目负责人昨日发推暗示，Windows 10 手机预览版可能会在本周推送，这对不少正在焦急等待升级的用户来说显然是个好消息。尽管更新推送近在眼前，但目前仍有不少用户对升级系统存在顾虑。

微软目前尚未发布系统更新支持的机型列表以及相关的更新条件，因此许多仍未获得 Lumia Denim 更新的用户担心会自己的设备会不支持系统升级。

对于这个问题，微软 Windows Insider 项目负责人 Gabriel Aul 今天在回答网友提问“ Lumia Denim 更新是否会影响 Win10 手机预览版升级”的问题时明确表示，Lumia Denim 并不是升级 Win10 手机预览版更新的必要条件，大家无需为此担心。

也就是说，即使你的设备现在运行的仍然是Lumia Denim之前的系统版本，只要你注册成为 Windows Insider 手机预览版内测会员，也有可能会获得 Win10 手机预览版的更新推送。不过，我们留意到微软依旧没有提到升级 Windows 10 手机版的最低配置要求或是系统版本要求，因此具体怎样才算“有资格”升级依旧未知。
