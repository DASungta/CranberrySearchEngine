微软推Cloud Platform线路图网站：专注云端产品更新讯息

0 cnBeta.COM   2015-02-01 08:30:01

摘要：据外媒报道，日前，微软推出了一个Cloud Platform线路图网站，专用于分享云端相关技术产品的发布时间。过去几年时间里，微软曾大力开发云端产品，使其成为一位相当具有竞争力的云端技术供应商。现在，为了让用户可以更好地了解自家的云端产品，这家公司专门上线了一个与之相关的线路图网站。

据悉，微软现在拥有大量的云端产品，如Azure、Intone、Power BI、Visual Studio Online。Cloud Platform涵盖了所有相关联的产品，除了上面提到的这几个，像Windows Server、System Center、SQL Server、Visual Studio也都在列。除此之外，Cloud Platform System、Analytics Platform System、StorSimple同样也都能在该网站上找到更新日期。
