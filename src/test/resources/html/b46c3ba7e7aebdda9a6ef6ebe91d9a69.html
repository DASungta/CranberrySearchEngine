尼克罗顿今年2月将推出面向儿童的独立视频订阅服务

0 cnBeta.COM   2015-01-30 08:54:56

摘要：主打儿童市场的娱乐巨头尼克罗顿（Nickelodeon）近日透露将效仿HBO电台在今年2月份推出直接面向消费者的订阅服务。尼克罗顿的母公司美国维亚康姆集团（Viacom）的首席执行官Philippe Dauman在本周四早晨的投资电话会议中透露了这条消息，关于该项服务的更多细节将会在下月由尼克罗顿召开的同广告商的前期会议中公布。

这款新推出的尼克罗顿服务声称主要面向那些使用移动设备的消费群体，Dauman表示这项服务将会拥有不同的名称或者品牌，在推出之后对于父母和青少年将会非常有吸引力。
