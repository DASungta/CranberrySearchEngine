Windows 8.x今年一月份市场份额不升反降

32 cnBeta.COM   2015-02-01 21:40:59

摘要：网络分析公司NetMarketShare发布了每月桌面操作系统使用份额统计数据，显示Windows8.x的份额大幅暴跌，而一天后该公司公布的修正数据，显示Windows8.x的份额仍然大幅度下降，但不是很严重。tMarketShare去年12月份的统计数据显示，Windows8.x市场份额是13.52%，比去年11月下降5.13个百分点，已经低于Windows XP的市场份额。

NetMarketShare最新统计数据显示，Windows 8的市场份额从去年12月份的4.03%下降到1月份的3.79%，跌幅0.24个百分点。Windows 8.1的市场份额从去年12月份的9.49%上升到到1月份的10.04%，升幅0.5个百分点。Windows8.x总市场份额仅增长0.03个百分点。

相比之下，Windows XP今年1月份市场份额从去年12月的18.26%上升到18.93%，升幅在0.67%。Windows 7市场份额从去年12月的56.26%下降到今年1月份的55.92%，跌幅0.34个百分点。
