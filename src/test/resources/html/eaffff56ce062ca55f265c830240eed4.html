一些用户劝微软 别杀Windows RT再给次机会

27 威锋网   2015-02-10 17:34:49

摘要：2012 年 10 月份，微软正式发布了旗下新一代操作系统 Windows 8，声称专门针对触摸屏进行优化，可以在平板电脑等多款配备触摸屏的设备 上使用，试图通过一个系统占领所有设备。令人不解的是，微软竟然大张旗鼓地介绍了一个莫名其妙的 Windows 版本：Windows RT，并且还发 布了一款已经失败的参考设备 Surface RT。


Windows RT 发布之后吐槽声一片，相关设备也是亏本买卖，微软逐渐意识到了 Windows RT 的失败。所以我们很长时间已经没有看到微软提及 Windows RT 了，转而专注于完整的 Windows 体验。如按照这个方向发展，Windows 10 发布之后微软可能考虑完全放弃 Windows RT，包括提供的升级服务。对此，有一些用户表示不服，认为微软应该做出保证 Windows RT 活着而不是不管的承诺。

近日，在微软官方反馈平台 Windows UserVoice 上，一个有关 Windows RT 的主题迅速火热，并且投票给支持 Windows RT 活着的用户不少。发表该主题的用户认为，Windows RT 应该可以得到第二次机会，因为 Surface RT 和 Surface 2 两款平板电脑在全球也拥有上百万的用户，同时两款平板的性能并不差。

“尺寸大于 8 英寸基于 ARM 的平板电脑不能没有未来啊，微软请让 Windows RT 升级到 Windows 10 的平板电脑或智能手机版本吧。” 帖子发布者表示，“当然，Surface RT/2 拥有很漂亮的硬件配置，让其拥有完整的 Windows 10 功能和服务总比没有桌面的纯平板电脑更好。”

Windows RT 只能体验少数 Windows 10 功能

按照微软的计划，Windows 10 最终版秋季正式发布之后，才会为 Windows RT 提供少量功能上的改进升级。微软的意图非常明显，希望所有消费者都去购买能够运行完整版 Windows 10 操作系统的新平板电脑，如此才能体验所有新公布的功能和服务。

微软已经证实 Windows RT 将无法升级至完整版本的 Windows 10，功能上进行阉割，但到目前为止尚未公布详细细节，包括具体原因。有可能因为新一代操作系统 Windows 10 还处于开发阶段，在最终版正式推出之前或许很多功能还会发生改变， 也有可能是因为 ARM 与 x86/x64 芯片不同而造成的限制。

此前消息报道称，微软已经停产 Surface 2 和 Lumia 2520 两款平板电脑，而且微软至今也未能解决用户对 Windows RT 的误解。更重要的是，去年一整年，我们看到的是可完整运行 Windows 的 Surface Pro 3，而 Surface 2 的升级版 Surface 3 完全毫无音讯，如果 Windows RT 真的有前途，微软不会不准备新的硬件设备。

总之，Windows RT  生存下来的机会已经微乎其微，而人们对 Windows RT 的印象也会在接下来几年时间里逐渐淡忘。
